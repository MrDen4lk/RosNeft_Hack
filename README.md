# RosNeft_Hack

1. Неоюходимо решить задачу сегментации на 40 классов
2. Были дан датасет с картинками разрешением 600х600, всего 10к семплов
3. Изображения представляли из себя схему с условными обозначениями
4. Была выбрана архитекутра Unet++ с бекбоном Efficientnet_b0, а также использовался self-attantion
5. Loss функция была выбрана DiceCE
6. Основное метрика выбрана IoU без учета фона (class=0)
7. Была попытка обучить SegFormer, но из-за небольшого количества данных и специфичного вида, точность была хуже
8. Итоговая модель на валидации показала метрику IoU=0,96
9. Также был сделан инференс на ONNX
10. Обучение производилось на Nvidia RTX 5070 ti
